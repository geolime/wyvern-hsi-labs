# Wyvern Hyperspectral Analysis Projects

This repository contains a collection of small, focused hyperspectral remote sensing projects using Wyvern Open Data.

Each project explores a different analysis task and dataset.

---

## Projects

### 1) Forest and Agricultural Land Cover Mapping

Location: Santa Cruz de la Sierra, Bolivia

Techniques:
- PCA + KMeans clustering
- Spectral Angle Mapping (SAM)
- NDVI and red-edge spectral validation
- PCA visualization composites

➡ View project: [Forest Land Cover Mapping](projects/forest_landcover)

---
### Water Quality — Bitter Lake (Egypt)

Hyperspectral water analysis using Wyvern Open Data.  
Includes turbidity and chlorophyll proxy mapping, spectral feature analysis, and unsupervised optical water-type clustering.


➡ View project: [Water Quality Mapping](projects/water_quality)
